## Sistema de Asignación de Jurados

### Algoritmo Inteligente

**Criterios de Asignación:**
1. **Especialización**: Coincidencia con tipo de medio
2. **Experiencia**: Años de experiencia en el área
3. **Certificaciones**: Validación de credenciales
4. **Carga de Trabajo**: Balanceo entre jurados
5. **Conflictos**: Evitar evaluación de conocidos

**Implementación:**
```typescript
class JuradoAssignmentService {
  async asignarJuradosInteligente(concursoId: number): Promise<AsignacionResult> {
    // 1. Obtener categorías del concurso
    const categorias = await this.getCategoriasByConcurso(concursoId);
    
    // 2. Para cada categoría, encontrar jurados especializados
    const asignaciones: Asignacion[] = [];
    
    for (const categoria of categorias) {
      const tipoMedio = this.detectarTipoMedio(categoria.nombre);
      const juradosDisponibles = await this.getJuradosEspecializados(tipoMedio);
      
      // 3. Aplicar algoritmo de asignación
      const juradosSeleccionados = this.seleccionarJurados(
        juradosDisponibles,
        categoria,
        3 // Mínimo 3 jurados por categoría
      );
      
      // 4. Crear asignaciones
      for (const jurado of juradosSeleccionados) {
        asignaciones.push({
          jurado_id: jurado.id,
          categoria_id: categoria.id,
          fecha_asignacion: new Date()
        });
      }
    }
    
    return { asignaciones, total: asignaciones.length };
  }
}
```
