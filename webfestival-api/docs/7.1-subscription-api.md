# API de Gestión de Suscripciones - WebFestival

## Descripción General

La API de gestión de suscripciones proporciona endpoints completos para manejar planes de suscripción, límites de usuario, upgrades y tracking de uso en la plataforma WebFestival.

## Endpoints Implementados

### 📋 Gestión de Planes

#### `GET /api/v1/subscriptions/plans`
Obtiene todos los planes de suscripción disponibles.

**Parámetros de consulta:**
- `active` (boolean, opcional): Filtrar solo planes activos
- `interval` (string, opcional): Filtrar por intervalo (`monthly`, `yearly`)

**Respuesta exitosa (200):**
```json
{
  "success": true,
  "data": [
    {
      "id": "basico",
      "name": "Plan Básico",
      "price": 0,
      "currency": "USD",
      "interval": "monthly",
      "features": [
        {
          "key": "contests",
          "name": "Participación en concursos",
          "description": "Participa en concursos públicos",
          "enabled": true
        }
      ],
      "limits": {
        "maxConcursosPerMonth": 5,
        "maxUploadsPerMonth": 10,
        "maxPrivateContests": 0,
        "maxTeamMembers": 0,
        "analyticsAccess": false,
        "prioritySupport": false,
        "apiAccess": false
      },
      "active": true,
      "created_at": "2025-01-01T00:00:00.000Z"
    }
  ],
  "message": "Planes de suscripción obtenidos exitosamente"
}
```

#### `GET /api/v1/subscriptions/plans/:planId`
Obtiene un plan específico por ID.

**Parámetros:**
- `planId` (string): ID del plan a obtener

**Respuestas:**
- `200`: Plan encontrado
- `404`: Plan no encontrado

### 🔐 Gestión de Límites y Permisos

#### `GET /api/v1/subscriptions/limits`
Obtiene los límites de uso del usuario autenticado.

**Autenticación:** Requerida

**Respuesta exitosa (200):**
```json
{
  "success": true,
  "data": {
    "canParticipateInContest": true,
    "canUploadMedia": true,
    "canCreatePrivateContest": false,
    "canAddTeamMember": false,
    "hasAnalyticsAccess": false,
    "hasPrioritySupport": false,
    "hasApiAccess": false,
    "remainingConcursos": 5,
    "remainingUploads": 10,
    "remainingPrivateContests": 0,
    "remainingTeamMembers": 0
  },
  "message": "Límites de uso obtenidos exitosamente"
}
```

#### `GET /api/v1/subscriptions/can/:action`
Verifica si el usuario puede realizar una acción específica.

**Autenticación:** Requerida

**Parámetros:**
- `action` (string): Acción a verificar (`participate`, `upload`, `create_private`, `add_member`)

**Respuesta exitosa (200):**
```json
{
  "success": true,
  "data": {
    "canPerform": true,
    "action": "participate"
  },
  "message": "Permisos verificados exitosamente"
}
```

### 📈 Gestión de Suscripciones

#### `GET /api/v1/subscriptions/my-subscription`
Obtiene la suscripción activa del usuario autenticado.

**Autenticación:** Requerida

**Respuestas:**
- `200`: Suscripción encontrada
- `404`: No tiene suscripción activa

#### `POST /api/v1/subscriptions/upgrade`
Mejora la suscripción del usuario a un plan superior.

**Autenticación:** Requerida

**Cuerpo de la petición:**
```json
{
  "planId": "profesional"
}
```

**Respuestas:**
- `200`: Upgrade exitoso
- `400`: Datos inválidos
- `404`: Usuario sin suscripción activa
- `500`: Error interno

#### `POST /api/v1/subscriptions/cancel`
Cancela la suscripción activa del usuario.

**Autenticación:** Requerida

**Respuesta exitosa (200):**
```json
{
  "success": true,
  "message": "Suscripción cancelada exitosamente"
}
```

### 🔧 Endpoints de Administración

#### `POST /api/v1/subscriptions/plans`
Crea un nuevo plan de suscripción.

**Autenticación:** Requerida (ADMIN)

**Cuerpo de la petición:**
```json
{
  "id": "nuevo-plan",
  "name": "Nuevo Plan",
  "price": 29.99,
  "currency": "USD",
  "interval": "monthly",
  "features": [
    {
      "key": "feature1",
      "name": "Característica 1",
      "description": "Descripción de la característica",
      "enabled": true
    }
  ],
  "limits": {
    "maxConcursosPerMonth": 20,
    "maxUploadsPerMonth": 50,
    "maxPrivateContests": 5,
    "maxTeamMembers": 5,
    "analyticsAccess": true,
    "prioritySupport": true,
    "apiAccess": false
  }
}
```

#### `PUT /api/v1/subscriptions/plans/:planId`
Actualiza un plan de suscripción existente.

**Autenticación:** Requerida (ADMIN)

#### `DELETE /api/v1/subscriptions/plans/:planId`
Elimina un plan de suscripción (lo marca como inactivo).

**Autenticación:** Requerida (ADMIN)

#### `GET /api/v1/subscriptions/metrics`

#### `POST /api/v1/subscriptions/initialize-plans`
Inicializa los planes predeterminados en la base de datos.

**Autenticación:** Requerida (ADMIN)

### 💳 Procesamiento de Pagos

#### `POST /api/v1/subscriptions/process-payment`
Procesa un pago para una suscripción.

**Autenticación:** Requerida

**Parámetros de consulta:**
- `provider` (string, opcional): Proveedor de pago (`stripe`, `paypal`). Por defecto: `stripe`

**Cuerpo de la petición:**
```json
{
  "user_id": "user123",
  "plan_id": "profesional",
  "payment_method": {
    "type": "card",
    "card": {
      "number": "4242424242424242",
      "exp_month": 12,
      "exp_year": 2025,
      "cvc": "123"
    }
  }
}
```

#### `POST /api/v1/subscriptions/confirm-paypal`
Confirma una suscripción de PayPal después de la aprobación.

**Autenticación:** Requerida

### 🔄 Webhooks

#### `POST /api/v1/subscriptions/webhooks/stripe`
Maneja webhooks de Stripe para eventos de pago.

**Autenticación:** No requerida (verificación por firma)

#### `POST /api/v1/subscriptions/webhooks/paypal`
Maneja webhooks de PayPal para eventos de pago.

**Autenticación:** No requerida (verificación por firma)

## Planes Predeterminados

### Plan Básico (Gratuito)
- **ID:** `basico`
- **Precio:** $0/mes
- **Límites:**
  - 5 concursos por mes
  - 10 uploads por mes
  - 0 concursos privados
  - Sin analytics ni soporte prioritario

### Plan Profesional
- **ID:** `profesional`
- **Precio:** $19.99/mes
- **Límites:**
  - Concursos ilimitados
  - Uploads ilimitados
  - 2 concursos privados
  - 3 miembros de equipo
  - Analytics básicos y soporte prioritario

### Plan Premium
- **ID:** `premium`
- **Precio:** $49.99/mes
- **Límites:**
  - Todo ilimitado
  - 10 concursos privados
  - 10 miembros de equipo
  - Analytics avanzados y acceso API

### Plan Organizador
- **ID:** `organizador`
- **Precio:** $99.99/mes
- **Límites:**
  - Todo ilimitado
  - Concursos privados ilimitados
  - 50 miembros de equipo
  - Funciones empresariales

## Códigos de Error

- `400`: Datos de entrada inválidos
- `401`: No autenticado
- `403`: Sin permisos suficientes
- `404`: Recurso no encontrado
- `409`: Conflicto (ej. plan ya existe)
- `500`: Error interno del servidor

## Validación de Datos

Todos los endpoints utilizan validación con Zod schemas para garantizar la integridad de los datos. Los errores de validación incluyen detalles específicos sobre los campos inválidos.

## Seguridad

- Autenticación JWT requerida para endpoints protegidos
- Verificación de roles para endpoints administrativos
- Validación de firmas para webhooks
- Rate limiting implementado
- Sanitización de datos de entrada

## Testing

Los endpoints están completamente probados con:
- Tests unitarios para lógica de negocio
- Tests de integración para flujos completos
- Mocks para servicios externos (Stripe, PayPal)
- Validación de permisos y autenticación
- Casos de error y edge cases

## Ejemplos de Uso

### Obtener planes disponibles
```bash
curl -X GET "https://api.webfestival.com/v1/subscriptions/plans"
```

### Verificar límites del usuario
```bash
curl -X GET "https://api.webfestival.com/v1/subscriptions/limits" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Hacer upgrade de plan
```bash
curl -X POST "https://api.webfestival.com/v1/subscriptions/upgrade" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"planId": "profesional"}'
```

## Notas de Implementación

- Los límites se verifican en tiempo real antes de permitir acciones
- El tracking de uso se actualiza automáticamente
- Los webhooks manejan renovaciones y cancelaciones automáticamente
- Los planes se pueden activar/desactivar sin eliminar datos
- Soporte para múltiples monedas (configuración por plan)
